# 📈 AI Stock Market Price Prediction Dashboard

An interactive AI-powered Stock Market Price Prediction Dashboard built using Machine Learning, Deep Learning, and Streamlit.

## 🚀 Project Overview

This project predicts and visualizes stock market trends using historical stock price data. It combines data analysis, machine learning models, and an interactive web dashboard to help users explore stock performance.

## ✨ Features

- 📊 Interactive Stock Price Dashboard
- 📅 Date Range Filter
- 📈 Interactive Plotly Charts
- 📥 Download Dataset as CSV
- 🤖 XGBoost & LSTM Model Selection (Project Structure Ready)
- 🎨 Professional Streamlit Dashboard
- 📉 Historical Price Visualization
- 💻 User-Friendly Interface

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- TensorFlow / Keras
- XGBoost
- Joblib
